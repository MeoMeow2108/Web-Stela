

## Tùy chỉnh

### Thay đổi thông tin server
- Mở file `index.html` và tìm các thông tin về server
- Thay đổi IP server, tên server, thông tin thanh toán


### Thay đổi màu sắc
- Mở file `style.css`
- Tìm section `:root` và thay đổi các biến màu
- Các biến chính:
  - `--primary`: Màu chính
  - `--secondary`: Màu phụ
  - `--accent`: Màu nhấn
  - `--dark`: Màu nền tối

### Thêm sản phẩm mới
- Mở file `index.html`
- Tìm section `Rank Packages` hoặc `Items/Keys`
- Thêm card mới theo mẫu có sẵn

## Mock Data

Website sử dụng mock data cho:
- Server status: Random online/offline
- Discord members: Số liệu giả
- Stats: Dữ liệu thống kê giả
- Black market listings: Lưu trong localStorage

Trong thực tế, bạn có thể kết nối với backend để lấy dữ liệu thật.

## Hỗ trợ

Nếu có bất kỳ vấn đề gì khi triển khai hoặc cần hỗ trợ thêm, vui lòng liên hệ qua Discord server của STELA SMP.
